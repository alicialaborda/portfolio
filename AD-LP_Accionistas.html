ADOLFO DOMINGUEZ LANDING PAGE ACCIONISTAS

Case

Adolfo Domínguez needs to develop a process that allows to collect personal information and consents of shareholders.
The purpose is to be able to incorporate the shareholder data into the Salesforce universe and to identify all sharehorders as such with a boolean field.
Adolfo Domínguez will send a card via postal mail that will include a QR code that will take the shareholders to the Landing Page that will be implemented in SFMC.

Solution

All processing of what to display and the update of attributes (consent/s and date/s) in the CRM will be developed using AMPScript code.

Programación Web · Salesforce Marketing Cloud









Error Control

The URL will be personalize for each shareholder, including a unique code to identify them in the landing. If this code is not informed or is not in our data base, the landing page will display an error message.
