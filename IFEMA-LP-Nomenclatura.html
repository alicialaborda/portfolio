IFEMA LANDING PAGE

Case

IFEMA Madrid needs to develop a process to generate the naming of communications for traking purposes.

Solution

The first CloudPage will identify the user inside SFMC, if it is not in the Data Base of users allowed to access the Landing Page, it will display a KO. If the person is in the Data Base it will display the second CloudPage to create the naming for the communication. Once the form is filled with all the informtion regarding the send, they will confirm and get the naming they will have to use for the communication.
