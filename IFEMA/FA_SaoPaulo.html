FRUIT ATTRACTION SAO PAULO LANDING PAGE

Case

Fruit Attraction Sao Paulo needs to develop a process that allows to collect consents for their new communications.
The purpose is to convert as much contacts into subscribers as possible and to incorporate the data into the Salesforce Marketing Cloud.
Fruit Attraction Sao Paulo will send an email that will include a subscribe button with ampscript code that will take the contact to the Landing Pages implemented in SFMC in order to save the information of the new subscriber.

Solution

All processing of what to display and the update of attributes (consent/s and date/s) will be developed using AMPScript code.

Programación Web · Salesforce Marketing Cloud









Error Control

The URL will contain parameters from the email in order to identify the contact in la Landing Page. If this data is not informed or is not in our data base, the landing page will display an error message.





The form in the LP contains code to validate that the mandatory fields are completed.





If the person has already completed the form, the landing page will display a "You have already filled the form" error message.
